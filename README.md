<H1> Titulo del proyecto: Dataset de ventas simuladas.</H1>

<H2>Escenario elegido</H2>
Se eligio el escenario Dataset de ventas simuladas. Punto 10.3 en la pagina 8 del TP. Se utilizo el dataset provisto para hacer un analisis en el cual se muestra: 
<ul>
<li>la cantidad de ventas totales en el año</li>
<li>la cantidad totales de ventas por mes</li>
</ul>

Se crea un grafico de barras que muestra cuantas ventas se hicieron por mes y un archivo csv con la cantidad de ventas por mes, estos archivos son guardados en la carpeta resultados.

<H2>Descripcion del dataset utilizado</H2>
El data set contiene 3 columnas: <b>ID, sales_date, sales_amount</b>. Las columnas sales_date y sales_amount son las utilizadas en el proyecto, sales_date corresponde al dia y sales_amount a la cantidad de ventas realizadas en ese dia.

<H2>Objetivo</H2>
Analizar los datos provistos por el CSV para transformarlos en informacion relevante que pueda ser utilizada para tomar decisiones.

<H2>Packages necesarios</H2>
<ul>
  <li>Pandas</li>
  <li>csv</li>
  <li>matplotlib</li>
</ul>
