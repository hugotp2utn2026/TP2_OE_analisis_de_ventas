<H1> Titulo del proyecto: Dataset de ventas simuladas.</H1>

<H2>Escenario elegido</H2>
Se eligio el escenario Dataset de ventas simuladas. Punto 10.3 en la pagina 8 del TP. 

Se crea un grafico de barras que muestra cuantas ventas se hicieron por mes y un archivo csv con la cantidad de ventas por mes, estos archivos son guardados en la carpeta resultados.

<H2>Objetivo</H2>
Analizar los datos provistos por el CSV para transformarlos en informacion relevante que pueda ser utilizada para tomar decisiones.
